Factors and Divisors
Factoring properties
Factoring is an extremely important tool in competition math. It requires you to understand many different properties, and these properties will be taught in this lesson. The first property is on the number of divisors of a given number. To find the number of divisors, you must first how many different prime factors there are of a number. For example, the number 84 can be represented as 2^2 * 41. From here, you must add one to the number of each prime factor, in this case being (2+1)*(1+1) as there are two 2 factors and one 41 factor. This would give us the final result of 6 total factors of the number 84.
Factors using quadratics
Divisibility rules can be extremely helpful in competition math as you are under time pressure. Due to this time pressure, students are forced to think quickly, especially in complex number theory problems. Therefore, by knowing these properties, not only will you be able to solve more problems but will be able to do so much faster, giving you the edge in the actual competition. For example, when asked for three unknown digits in the number 13, you can use divisibility rules to fill in the gaps and actually solve the problem using pure intuition.
